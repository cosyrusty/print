# print
Cat command in rust. Prints each file to stdout.

## Usage
Concatenate FILE(s) to standard output.
With no FILE, or when FILE is -, read standard input.

- using binary
  ```
  print [FILE]...
  ```
- read stdin as input to print
  ```
  print -
  ```
- using cargo
  ```
  cargo run -- [FILE]...
  ```
